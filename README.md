# 🤖 Telegram + GPT-4o-mini 聊天机器人

## 功能
- 支持 Telegram 对话
- 自动调用 GPT-4o-mini 生成回复
- 可在本地（macOS）直接运行

## 环境变量
请在终端中设置：
```bash
export TELEGRAM_TOKEN=""
export OPENAI_API_KEY="你的 OpenAI Key"
```

## 安装依赖
```bash
pip install -r requirements.txt
```

## 启动机器人
```bash
python bot.py
```

## 使用
打开 Telegram，找到你的机器人，输入 `/start` 或发送任意消息即可与 GPT 对话。
